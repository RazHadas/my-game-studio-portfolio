# Crypto Clicker Prompts Log

This file contains the history of prompts used to generate and improve the Crypto Clicker game.

## Prompt 1: Initial Game Creation
> Create and iteratively enhance a modern 'Crypto Clicker' game in a single HTML file. Include a clickable Bitcoin button, a 'Total Crypto' counter, and a shop panel with upgrade items. Use a dark mode design with neon green accents.

## Prompt 2: Mechanics and Visuals
> Ensure the price of shop items increases by 15% after each purchase. Add an auto-save and load feature using LocalStorage. Change the main clickable coin from an SVG to a generated transparent image for a more modern look.
