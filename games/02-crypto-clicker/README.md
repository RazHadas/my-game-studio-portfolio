# Crypto Clicker

Mine crypto, buy upgrades, and become a tycoon in this addictive clicker idle game.

## Structure
- `/src`: Contains the source code (HTML, CSS, JS)
- `/assets`: Contains game assets (images)
- `prompts_log.md`: Log of AI prompts used to create the game

## How to Play
- Click the Bitcoin to earn crypto.
- Use your crypto to buy upgrades from the marketplace.
- Upgrades like GPUs and Mining Farms automatically mine crypto for you over time.
- Your progress is saved automatically.
