# Neon Snake

A modern retro twist on the classic snake game. Collect food and grow without hitting the walls!

## Structure

- `/src`: Contains the source code (HTML, CSS, JS)
- `/assets`: Contains game assets (sprites, images)
- `prompts_log.md`: Log of AI prompts used to create the game

## How to Play

Use arrow keys to control the snake. Eat food to grow and earn points. Avoid hitting the walls or yourself.
